Welcome!

AnimaniaJS is an open-source jQuery plugin, which makes 3 simple animations, namely: float, puff and credits animations simpler. The main aim of the plugin is to ease the handling of often complex jQuery.animate() calls and provide customization options to the user.

Getting started

You can get cracking with AnimaniaJS by including the following two lines inside the <head> tag or at the end of your body content.

<script type='text/javascript' src='http://code.jquery.com/jquery-1.11.2.min.js'></script>

Minified version: <script type='text/javascript' src='https://cdn.rawgit.com/samrith-s/AnimaniaJS/v1.0.1/animania-1.0.1.min.js'></script>
Complete version: <script type='text/javascript' src='https://cdn.rawgit.com/samrith-s/AnimaniaJS/v1.0.1/animania-1.0.1.js'></script>

Optionally, you can also include jQuery migrate for better compatibility and warnings:

<script type='text/javascript' src='http://code.jquery.com/jquery-migrate-1.2.1.min.js'></script>

Please do note that including jQuery (and jQuery migrate) BEFORE the plugin script is necessary as this plugin extends jQuery functions.

Animations

Animations using AnimaniaJS are very, very easy to use. It requires just a basic understanding of JavaScript. Best part? It has a variety of customization options, and more keep on getting added with each update!

The plugin has a simple initialization call: $(object).animania("animation", options, callback()), which focuses on simplicity and customisability. You can choose not to explicitly mention any options or a callback. Or mention any either of them. Note: Mentioning the type of animation is vital for the plugin to function!

Important: As of 1.0.1, AnimaniaJS only performs animations on objects that are absolutely positioned. Future releases will most likely include support for relatively positioned elements as well.

The plugin offers a variety of options. There are a couple of global options that you need to set for all animations. These determine the transition of the animation and the duration. There is still some debate on the developer end to ease options further and make all of them global and extend more options per animation. As of now, each plugin has some options which can be set. Below is a table of the list of global animation options.

Option Value Description
duration Integer Animation duration in milliseconds
transition String The type of transition: entry or exit. Puff and Float animations require this value.

Puff

The puff animation is a simple animation with both entry and exit transitions. It automatically toggles display state of the object and readies it for the respective transition. Below is a basic example of a puff animation.

Select a transition:

This is a test div!

Example source code: 


$("#myDiv").animania("puff", {
	duration: 2000,
	transition: "entry",
	puff: {
		offset: 100
	}
});

Puff animation has it's set of option(s) which offer setting offset.

Option Value Description
offset Integer Offset from current position to set as the start or end point of animation

Float

The float animation is a brilliant animation with both entry and exit transitions as well as directions: up, down, left and right. It automatically toggles display state of the object and readies it for the respective transition and direction. Below is a basic example of a float animation.

Select a transition: and direction:

This is a test div!

Example source code: 


$("#myDiv").animania("float", {
	duration: 2000,
	transition: "entry",
	float: {
		direction: "up",
		offset: 50
	}
});

Float animation has it's set of option(s) which offer setting offset and direction.

Option Value Description
offset Integer Offset from current position to set as the start or end point of animation
direction String Direction of transition: up, down, left or right

Credits

The credits animation simply marquees the object either up or down. It also has built in support for animation pausing and slow drag on restart, which is toggled on mouseover and mouseout It automatically toggles display state of the object and readies it for the respective transition. Below is a basic example of a credits animation.

This is a test div!

Example source code: 


$("#myDiv").animania("credits", {
	duration: 2000,
	credits: {
		direction: "up",
		offset: 100
	}
});

Credits animation has it's set of option(s) which offer setting offset and direction.

Option Value Description
offset Integer Offset from current position to set as the start or end point of animation
hoverStop Boolean Decides if the animation should stop on mouse hover or continue as per usual
